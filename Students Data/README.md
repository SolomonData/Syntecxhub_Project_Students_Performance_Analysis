# Students Performance Analysis

## Project Overview

This project analyzes Students Performance to uncover students behaviour and insights for improving academic results.

## Tools Used

- Power Bi
- Power Querry
- Excel

## Problem Solved

- This project addresses the problem of monitoring and analyzing students performance across different conditions by providing a centralized dashboard for tracking these conditions.
